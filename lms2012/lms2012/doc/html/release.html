Release Firmware Package Check List
  1. Check version and debug/special settings in "lms2012.h"

  2. Check version in "bytecodes.h"

  3. Check version and change log in "Implementation"

  4. Build documentation

  5. Check documentation

  6. Update byte code compile ("cd ~/projects/lms2012/open_first", "./asm_update.sh"

  7. Update all graphics ("cd ~/projects/lms2012/lms2012/doc/graphics", "./update")

  8. Compile all bytecode programs ("cd ~/projects/lms2012/lmssrc", "./update_all")

  9. Compile project for Linux_AM1808

  10. Check that VM runs on NFS

  11. Make image ("cd ~/projects/lms2012/open_first", "sudo ./make_image.sh"

  12. Rename EV3-image.xxxxxxxxxx.bin to ev3_firmware_vxxx

  13. Update a bricks firmware with the new one

  14. Check that VM runs from the image

  15. Commit all changes to CVS

  16. Backup CVS version

  17. Compress project folder "~/projects/lms2012" into "lms2012_vxxx_yymmdd.tar.gz"

  18. Move "lms2012_vxxx_yymmdd.tar.gz" to a windows PC

  19. Move documentation folder "~/projects/lms2012/lms2012/doc/html" to a windows PC

  20. Rename old documentation "\\dka022\GIM_electronics\People\Tommy Christian Pedersen\lms2012\html" to "...\html_v..."

  21. Copy documentation to "\\dka022\GIM_electronics\People\Tommy Christian Pedersen\lms2012\html"

  22. Add "lms2012_vxxx_yymmdd.tar.gz" to SourceSafe

  23. Upload "lms2012_vxxx_yymmdd.tar.gz" to PICO ftp site

  24. Send "html" in mail to NI and PICO and announce update
